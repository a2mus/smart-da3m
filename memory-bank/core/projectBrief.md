# Project Brief

## What We Are Building

**إحسان (Ihsane)** — A web-based adaptive learning platform specialized in intelligent pedagogical remediation for the Algerian primary education system (ages 6–11). The platform detects learning gaps through adaptive diagnostic testing and delivers personalized remediation pathways aligned with Algeria's national curriculum. It combines pedagogical rigor, emotional safety, spiritual values, and peer solidarity in a unique approach that no existing platform offers for the Arab/African market.

## Goals
- Validate the adaptive diagnostic + remediation engine with a pilot of < 500 students in 1–2 Algerian schools
- Prove measurable learning improvement (≥ 40% gap reduction rate)
- Deliver parent-facing dashboards that drive engagement (≥ 60% weekly access)
- Build a content back-office usable by non-technical pedagogical experts
- Collect analytics data sufficient to build a ministry partnership case (P2IA model)

## Non-Goals
- Live teacher/tutoring functionality (this is an expert system)
- Gamification, P2P learning, AI tutor (deferred to Beta v2)
- Mobile native apps (MVP is web-only SPA)
- Tamazight/English language support (MVP is Arabic + French bilingual)
- Voice recognition / ASR features
- Multi-school / multi-wilaya deployment

## Success Criteria
- Gap Reduction Rate ≥ 40%
- Retention Rate ≥ 70% after 1 week
- Parent dashboard engagement ≥ 60% weekly
- Session abandonment ≤ 30% (frustration metric)
- System uptime ≥ 99.5%
- Expert adoption: 100% of pilot experts create content via back-office
